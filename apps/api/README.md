# @liftivity/api

Fastify-based Node.js backend for Liftivity.

## Dev

```bash
pnpm dev        # ts-node-dev watch mode on http://localhost:3000
pnpm build      # compile → dist/
pnpm start      # run compiled output
```

## Endpoints

| Method | Path | Description |
|--------|------|-------------|
| GET | `/health` | Health probe |
