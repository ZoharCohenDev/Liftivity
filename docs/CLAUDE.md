The Liftivity folder contains everything needed to understand and build the system end-to-end: it includes user flow and UI mockups (how the product looks and behaves), MVP architecture and API contracts (how the system works behind the scenes), database schema and mock data (how data is structured), test scenarios and Definition of Done (how features are validated), and planning documents like build order and an AI roadmap that outline the development process and future direction 🚀
