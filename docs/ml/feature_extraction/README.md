# docs/ml/feature_extraction

Python scripts documenting and prototyping the feature extraction logic used by the AI services.

These are reference implementations — the production versions live in the AI services under `AI Services /`.

## Files

- `extractor.py` — main feature extractor entry point
- `dom_features.py` — DOM structure and semantic HTML feature extraction
- `nlp_features.py` — text and NLP feature extraction
- `performance_features.py` — web performance metric feature extraction
