# nlp-service — features

This folder needs to be implemented with text feature extraction logic for the NLP service.

## What to implement

- `readability.py` — Flesch-Kincaid, SMOG, and other readability score calculations
- `sentiment.py` — sentiment analysis on headlines, CTAs, and body copy
- `cta_extractor.py` — identify and score call-to-action phrases and button labels
- `keyword_density.py` — keyword frequency, topic relevance, SEO signal extraction
- `headline_analyzer.py` — headline clarity, emotional trigger words, length scoring
- `base.py` — abstract base class for all text feature extractors
