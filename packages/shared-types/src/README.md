# packages/shared-types/src

TypeScript type definitions shared between the frontend and backend.

## What to implement

- `api.ts` — request/response shapes for all API endpoints
- `models.ts` — shared domain types (`User`, `Site`, `Analysis`, `Issue`, `Report`)
- `enums.ts` — shared enums (`AnalysisStatus`, `IssueSeverity`, `PlanType`)
- `index.ts` — barrel export
