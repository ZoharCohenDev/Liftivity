# components

Global reusable UI components used across multiple pages.

These components have no knowledge of specific pages or business domain — they are purely presentational building blocks.

## Current components

- `ScoreBadge` — colored badge displaying a numeric score
- `ScoreDonut` — donut chart visualization of a score
- `SectionHeader` — consistent section title + subtitle + optional action slot
- `StatCard` — metric card with label, value, and trend indicator
- `StatusBadge` — status chip (healthy / warning / critical)
- `ThemeToggle` — dark/light mode toggle button
- `UsageBar` — usage progress bar with label

## nav/

Navigation components used by layouts: `Sidebar` and `TopBar`.
