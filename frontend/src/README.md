# frontend — src

React + TypeScript source for the Liftivity web app.

## Structure

| Folder | Purpose |
|---|---|
| `components/` | Global reusable UI components (used across multiple pages) |
| `context/` | React Context providers — app state, theme |
| `data/` | Mock / static frontend data |
| `hooks/` | Shared custom React hooks |
| `layouts/` | Page layout wrappers (app shell, public layout) |
| `pages/` | Route-level pages, each with co-located components |
| `router/` | React Router configuration |
| `theme/` | MUI theme, color tokens, dark/light mode |
| `types/` | Shared TypeScript type definitions |
| `utils/` | Shared helper functions |
