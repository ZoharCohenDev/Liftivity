# pages/Dashboard

Main dashboard page shown after login.

## Files

- `DashboardPage.tsx` — route component at `/app/overview`

## components/

Page-specific components used only by the Dashboard:

- `ConversionChart` — area chart showing conversion trends over time
- `IssueBreakdownChart` — bar chart of issue counts by category
- `RecentAnalysesTable` — table of the user's latest site analyses
