# pages/Analysis/components

Components used exclusively by the Analysis pages. Do not import these from other pages.

- `AnalysisDepthSelector` — toggle between Quick / Standard / Deep analysis depth
- `EarlyInsights` — shows partial results while the analysis is still running
- `EngineConsole` — scrolling log console with color-coded engine output lines
- `RecentURLHistory` — shows the last N analyzed URLs with scores for quick access
- `StageCard` — displays one analysis stage with icon, name, status, and timing
