# pages/Report/components

Components used exclusively by `ReportPage`. Do not import these from other pages.

- `ActionableFixItem` — collapsible item with issue severity, description, and step-by-step fix
- `BreakdownCard` — category score card with progress bar and issue count
- `ConversionHealthCard` — top-level score summary with `ScoreDonut` visualization
- `TopRecommendations` — top 3–5 recommendations sorted by estimated conversion impact
- `WebVitalsChart` — LCP / FID / CLS / TTFB bar chart with pass/fail thresholds
