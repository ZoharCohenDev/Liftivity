# pages/Landing/components

Components used exclusively by `LandingPage`. Do not import these from other pages.

- `Hero` — main above-the-fold section with headline, subtext, and CTA button
- `FeatureGrid` — 6-card grid of product features loaded from `mockData`
- `Testimonial` — star-rated customer quote cards
- `Pricing` — side-by-side plan cards (Free / Pro / Enterprise) with feature lists
- `CallToAction` — full-width bottom section encouraging sign-up
- `LandingNav` — sticky top bar with logo, nav links, and theme toggle
- `LandingFooter` — site footer with columns of links
