# pages

Route-level pages. Each page lives in its own folder alongside the components it exclusively uses.

## Pages

| Folder | Route | Description |
|---|---|---|
| `Dashboard/` | `/app/overview` | Main dashboard — stats, charts, recent analyses table |
| `Analysis/` | `/app/new-analysis`, `/app/analysis-progress` | New analysis form and live progress view |
| `Report/` | `/app/report` | Full analysis report with scores, issues, and recommendations |
| `Landing/` | `/` | Public landing page — hero, features, pricing, CTA |
