# types

Shared TypeScript type definitions used across the frontend.

## What to implement

- `api.ts` — API response shapes (mirrors backend DTOs)
- `models.ts` — frontend domain types (`User`, `Site`, `Analysis`, `Issue`, `Report`)
- `theme.ts` — theme-related types (e.g. `ColorMode`)
- `common.ts` — generic utility types (`Nullable<T>`, `AsyncState<T>`, etc.)
